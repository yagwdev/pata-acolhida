# 🐾 Pata Acolhida

Este projeto é um site desenvolvido como parte do trabalho prático da disciplina de **Introdução à Ciência da Computação (ICC)** da **Universidade Federal do Espirito Santo (UFES)**. 

## 🎯 Objetivo do Projeto
A proposta do trabalho era identificar um problema real na sociedade ou uma necessidade de um grupo específico de pessoas e desenvolver uma solução tecnológica para ajudá-los. 

O **Pata Acolhida** foi criado com o objetivo de apoiar as **ONGs de proteção animal** e os **animais abandonados**. O site serve como uma ponte entre as instituições e as pessoas que desejam adotar um pet, facilitando o processo de adoção responsável, centralizando a listagem de animais disponíveis e oferecendo um painel administrativo para o gerenciamento das informações.

## 🛠️ Tecnologias Utilizadas
O site foi construído de forma nativa e com código limpo, utilizando as tecnologias básicas da Web:
- **HTML5:** Para a estruturação semântica das páginas.
- **CSS3:** Para a estilização visual, organização de layout (Flexbox) e responsividade.
- **JavaScript:** Para a lógica e interações da interface.
